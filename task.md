# レポート作成のタスクリスト

 ## タスクリスト
- [ ] 開発者向けドキュメントの作業
  - [ ] 対象とするシステムの決定
  - [ ] データ構造の決定
  - [ ] ページ構造の検討
    - [ ] 一覧表示
    - [ ] 詳細表示
    - [ ] 新規登録
    - [ ] 編集フォーム
  - [ ] ページ遷移のの検討
  - [ ] HTTPメソッドとリソース名一覧
    - [ ] 一覧表示
    - [ ] 新規登録処理
    - [ ] データ更新処理
    - [ ] 削除処理
  - [ ] ページ遷移図の作成
  - [ ] データフローズの作成
- [ ] 管理者向けドキュメントの構成
  - [ ] インストールから起動までの手順確認
    - [ ] node.jsのインストール
    - [ ] Expressのインストール
    - [ ] 起動方法
  - [ ] 起動できない場合
  - [ ] 終了方法
  - [ ] わかっている不具合
- [ ] 利用者向けドキュメント
  - [ ] システムへのアクセスと全体把握
    - [ ] アプリケーション開始URLにアクセスできること
    - [ ] 最初の画面で，管理対象データの一覧が確認できること
  - [ ] データ閲覧のタスク
    - [ ] 一覧画面から新規登録用のフォームへ移動できること
    - [ ] フォーム必要事項を記入し，サーバーへ送信することで新しいレコードを作成できる
  - [ ] データ編集のタスク
    - [ ] 詳細情報画面から編集フォームへ移動できること
    - [ ] フォーム内の既存データを修正し，更新ボタンを押すことで，該当レコードのデータを更新できること
  - [ ] データ削除のタスク
    - [ ] 一覧画像または詳細画像から特定のレコードを削除できること
    - [ ] 削除実行前に確認のメッセージが表示されること
